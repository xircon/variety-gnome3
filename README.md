# variety-gnome3

+ Create a directory under `~/.local/share/gnome-shell/extensions` `called variety_control@manjaro`

+ Clone into this directory 

+ Restart Gnome shell

+ Enable in tweaks.

The directory should look like:
```
├── extension.js
├── locale
<snip>
├── metadata.json
├── README.md
├── Resources
│   ├── poppy-active.svg
│   ├── poppy.svg
│   ├── poppy.svg.old
│   └── variety.svg
└── stylesheet.css
```

Based on poppy-menu:

https://extensions.gnome.org/extension/1279/poppy-menu/
